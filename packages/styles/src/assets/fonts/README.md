# Source Han Serif CN Fonts

These files are extracted from the official Adobe Source Han Serif simplified-Chinese release:

- Release: `2.003R`
- Package: `14_SourceHanSerifCN.zip`
- Source: `https://github.com/adobe-fonts/source-han-serif/releases/tag/2.003R`
- License: `SIL Open Font License 1.1`

Vendored weights for the default Deweyou UI typography contract:

- `SourceHanSerifCN-Regular.otf` for body text
- `SourceHanSerifCN-Medium.otf` for emphasis text
- `SourceHanSerifCN-SemiBold.otf` for title text
- `SourceHanSerifCN-Bold.otf` for strong emphasis

The package intentionally keeps `mono` content on the existing monospace stack.
