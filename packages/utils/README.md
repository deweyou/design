# @deweyou-design/utils

Framework-agnostic runtime utilities for Deweyou Design packages and consumers.

## Installation

```bash
npm install @deweyou-design/utils
```

## Usage

```ts
import {} from '@deweyou-design/utils';
```

This package is currently a placeholder for shared utilities consumed across `@deweyou-design/*` packages. Runtime helpers will be added here as cross-package needs emerge.

## Dependency Rules

- No React dependency — utilities must remain framework-agnostic.
- May be consumed by: `@deweyou-design/react`, `@deweyou-design/react-hooks`, and app surfaces.
