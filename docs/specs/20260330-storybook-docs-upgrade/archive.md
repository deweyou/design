# 归档：Storybook 文档升级与重组

**分支**：`20260330-storybook-docs-upgrade`  
**完成时间**：2026-03-30  
**类型**：docs

---

## 交付摘要

对 Storybook 进行了三项改进：（1）重组 story 分类结构，将所有 story 收敛到 "Foundations"（Color）和 "Components"（Button、Typography、Popover、Icon、Menu）两个顶级分类，消除旧的"Internal review"分类；（2）为所有组件的全部公开 props 补充 `argTypes`（英文描述、类型、默认值），所有 story 用户可见文本统一为英文；（3）将 `@storybook/*` 包升级到最新稳定版本，验证所有 story 升级后正常运行无回归。

---

## 关键决策

| 决策              | 选择                          | 理由                                     | 备选方案                     |
| ----------------- | ----------------------------- | ---------------------------------------- | ---------------------------- |
| story 语言        | 全部英文                      | 面向外部开发者的接入文档，英文为国际通用 | 中英双语                     |
| 分类结构          | Foundations + Components 两层 | 简洁可预测，类比主流组件库文档结构       | 更细粒度的多级分类           |
| `IconButton` 文档 | 独立于 `Button` 单独列出      | 两者 API 不同，共用同一表格会造成混淆    | 合并为 `Button` 的子组件文档 |
